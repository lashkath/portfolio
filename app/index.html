PROJECTS

Fetch


Web application that helps connect pet owners to their lost pets.

Completed as the Capstone Project for Portland Code School, users can upload photos and descriptions of potentially lost pets seen around the community, or perform a filtered search of recent pet sightings to help locate their lost pet.

Tools used: Backbone.js, Node.js, Google Maps API, Orchestrate, Express, jQuery, Cloudinary


Pong


Implementation of classic arcade game using Python.

Assignment for An Introduction to Interactive Programming in Python. Course offered through Coursera.org from Rice University in Houston, TX.

Tools used: Python, CodeSkulptor
